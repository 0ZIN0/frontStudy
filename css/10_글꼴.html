# font-family

  • font-family property로 원하는 글꼴을 선택할 수있다
  • 기본적으로는 해당 컴퓨터에 설치되지 않는 글꼴은 사용할 수 없다
  • 글꼴이 존재하지 않는 경우에 대비해 여러가지 예비용 글골들을 설정해 둘 수 있다
  • 접속한 사용자에게 글골을 설치하게 하는 것 보다는 CDN이라는 개념으 많이 사용한다
  • CDN : Content Delivery Network, 다른 회사에서 인터넷에 올려놓고 배포하는 이미지, 글꼴 등의 리소스들을 의미한다
