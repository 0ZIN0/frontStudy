# 상대 길이 단위

  • em : 해당 요소의 font-size에 비례한 크기 설정
  • rem : 현재 페이지 전체(html)에 설정된 font-size에 비례한 크기 설정
  • vw : viewport width에 비례한 크기 설정
    1vw는 화면 너비의 1%크기를 의미한다
  • vh : viewport height에 비례한 크기 설정
    1vh는 화면 높이의 1%크기를 의미한다
